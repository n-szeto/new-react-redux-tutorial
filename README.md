## Features

* uses:
  * React (create-react-app)
  * Redux
  * Redux Saga

## Installation

* `git clone git@github.com:taming-the-state-in-react/react-redux-hackernews.git`
* `cd react-redux-hackernews`
* `npm install`
* `npm start`
* visit http://localhost:3000/
